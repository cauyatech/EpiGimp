# EpiGimp
EpiGimp est une application web d’édition d’images inspirée du logiciel libre GIMP. Le projet vise à offrir une expérience utilisateur moderne et simplifiée permettant de réaliser des opérations de base de retouche photo (recadrage, filtres, réglages de couleur, dessin, gestion des calques) directement depuis un navigateur web.
